## Deck title
**Where outbound dials turn into revenue — and where they don't**

## One-sentence thesis
Meeting-to-won conversion (12.8%) — not top-of-funnel volume — is where the pipeline leaks; one firmographic band (6–20 employees) decides almost all of it, and that rolls up into a single lead-scoring rule that gates the dial list.

## Executive summary (SCR)
- **Situation:** 102,007 call records produced 5,755 booked meetings, 2,829 held, and 362 won deals across NorthStar Bookkeeping's outbound dialer operation.
- **Complication:** Win rate varies 12.7× across employee bands AND 43% of all losses never reach a real sales conversation (meeting cancellations).
- **Resolution:** Score every company on employee band + industry; only dial the sweet spot; treat meeting confirmation as its own funnel step; recompute weights quarterly from the same star schema.

## Slide 10 detail — the lead-scoring recommendation

The deck closes on a single artifact: a transparent, additive scorecard. Not a black-box model — a rule a sales manager can explain in two minutes and a BI analyst can re-fit each quarter from the dim/fact tables. The data is bimodal, so the scorecard is too: employee band and anti-ICP industry do almost all the work.

| Feature | Bucket | Points |
|---|---|---|
| Employee band | 6–20 | +50 |
| | 21–50 | +12 |
| | 1–5 | +3 |
| | 51+ | 0 |
| Industry | Consulting / Marketing / Transport (anti-ICP) | −40 |
| | Construction / Healthcare / Retail / Other Services | +8 |
| | all other industries | +5 |
| Company form | Corporation / Partnership | +3 |
| | LLC / Sole Proprietorship | 0 |
| Cancel history | 0 prior cancels | +5 |
| | 1 prior cancel | 0 |
| | ≥2 prior cancels | −15 |

**Score band → action**
- **Dial (≥ 50):** top-of-queue — 22.7% of the universe, covers 83% of historic wins
- **Nurture (5–49):** email only, no dialer time — 63.4% of the universe
- **Skip (< 5):** remove from active list — the 13.8% that's anti-ICP or sole-operator

A middle "maybe" tier is deliberately absent — the data has one decisive split, not a gradient, and the scorecard reflects that rather than inventing nuance. Weights are derived from segment lift vs. the 12.8% baseline, not fit by gradient descent — keeps the model auditable.
